# coinstar
A program to analyse bitcoin market value for a given date range

## Use in CLI ##

```
coinstar.py <opts>

  -s    --start  <start date> YYYY.MM.DD
  -e    --end    <end date>   YYYY.MM.DD
  -r    --raw                 Show raw data
  -h    --help                Show this help
```

For example:
```
coinstar.py -s 2021.01.01 -e 2021.01.31
```
